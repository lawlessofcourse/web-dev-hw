How to make a splitboard!

This is a how to on making an old snowboard into a splitboard!

A splitboard is simple and one of the most worry-free tools to access the backcountry with. All you need to make one is a little shop-class skill, the Voile DIY Split Kit, and an old board to sacrifice to the powder deities. Read up on how it’s done and you’ll be making remote pow turns in no time. –B.G. - Transworld Snowboarding

Before you begin gathering materials and making a shopping list check out the local stores near you and see if they can split the board for you! If you don't want to risk a snowboard for the project see if a local shop can split it for you and hook you up with hardware!

Materials Needed:

  1. Voile DIY Split Kit (voile-usa.com)
  2. Hack Saw
  3. 9/64″, 3/16″, 1/4″, 3/4″ Drill Bits
  4. 3/8″ Nut Driver
  5. #3 Phillips Screwdriver
  6. C-Clamps
  7. Tape Measure
  8. Edge File
  9. Center Punch
  10. Ball-Peen Hammer
  11. Epoxy
  12. Blue Loctite
  13. Varnish, Varathane, or Enamel Paint
  14. Circular Saw (preferably with a carbide blade)
